Archive structure

Main file - happyBoxLanding/index.html

Landing page theme structure:

  • Stylesheets:
    • happyBoxLanding/css/..
      • libs.min.css - the compiled file of stylesheets framework and js-libraries
      • main.css - theme styles file.
      • main.css.map - compressed theme stylesheet.
  • js-files:
    • happyBoxLanding/js/..
      • libs.min.js - a compressed file that stores the js-functionality of frameworks and js-libraries
      • libs.js - a file that stores the js-functionality of frameworks and js-libraries
      • main.js - js file which stores the js functions. This is editable.
  • Image files - happyBoxLanding/images/ - this folder contains image files, theme sprite-icons and separate svg-icons.
  • Font files - happyBoxLanding/fonts/ - this folder contains fonts for this theme.

How to use the theme

The theme is built on a self-written layout using the Bootstrap grid.

Sliders use the Swiper slider library. To get more information about Swiper see the main source https://swiperjs.com.

Select and input [type = number] styled JCF plugin. To get more information about JCF see the main source https://www.psd2html.com/js-custom-forms/.

We'll use the jquery.validate plugin to validate the form. For more information about this plugin, see the main source https://jqueryvalidation.org/. ATTENTION. This plugin is used to demonstrate how a form works in layout. When porting a given theme to CMS platforms or frameworks, there may be conflicts with built-in functions.
